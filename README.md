# Sistema de Empleados con Testing

Sistema de gestión de empleados que demuestra **herencia y polimorfismo** en Java, con pruebas unitarias usando JUnit.

## Clases
- `Empleados` - Clase base
- `Gerente` - Maneja sueldo y aumentos
- `Tecnico` - Controla horas extras
- `EmpleadoVentas` - Registra ventas y comisiones

## Tecnologías
- Java 11+
- JUnit 5
